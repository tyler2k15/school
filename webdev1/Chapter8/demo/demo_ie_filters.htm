IE Filter Styles

Internet Explorer supports a wide variety of filters that you can use to add interesting visual effects to elements in your Web pages. There are two types of filter styles: those associated with IE 4.0 and those associated with IE 5.5 and above. Choose the filter style you want to apply from the list box. You can then select the parameter values for the selected filter from the appropriate list boxes. The filter effect will be applied to the image in the Preview box each time you select a parameter value. The corresponding filter style is displayed in the Style box.
